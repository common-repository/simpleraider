== Description ==
SimpleRaider is a raid and DKP management plugin for WordPress released under the GNU GPL v3 license.

It has support for multiple games from the start like Runes Of Magic, World Of Warcraft, Aion etc.

Features:

* raid templates ( you don't have to create new raids with boring copy ‘n paste)
* raid scheduling
* multiple games support
* multiple characters for users (for each game mutiple characters)
* DKP management
* game plugin support to get item stats of other games


== Installation ==


1. Upload the content of the SimpleRaider package to the `/wp-content/plugins/simpleraider/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open SimpleRaider in the administration console and click "create database tables"
4. Go the options menu of simpleraider and enter the page id on which you want to display SimpleRaider
